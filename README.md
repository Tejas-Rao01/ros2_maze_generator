# Maze Generator Workspace

This repository contains the source code and resources for the Maze Generator application.

## Features

- Generate random mazes of customizable sizes
- Visualize maze generation algorithms
- Export mazes to various formats

## Getting Started

1. **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/maze_generator_ws.git
    cd maze_generator_ws
    ```

2. **Install dependencies:**
    ```bash
    # Example for Python
    pip install -r requirements.txt
    ```

3. **Run the application:**
    ```bash
    python main.py
    ```

## Folder Structure

- `src/` - Source code for maze generation
- `docs/` - Documentation and resources
- `tests/` - Unit tests
